{{ else }}
メンバーＰ {{ if group.feedback }} （ニックネーム：{{ group.p_label }}） {{ endif }} はあなたに送ったトークンの何％をメンバーＲに送ると予想していたと思いますか？

あなたの前に意思決定をしたメンバーＰはあなたの行動（メンバーＰから受け取ったトークンの何％をメンバーＲに送るか）を予想しました。その予想を推測してください。

{{ endif }}

このラウンドがあなたの報酬の計算に選ばれた場合、あなたの予想についてもトークンを獲得できます。 あなたの予想が正しいほど、獲得できるトークンが多くなります（最大120トークン）。 獲得トークンの決まり方に関心のある方は下の「獲得トークンの計算方法」をクリックしてください。

{{ if group.dictator_first }}

あなたの予想とメンバーＰの予想の差（予想誤差）を100分の1にして、二乗します。 その値に120を掛けた値を120から差し引いたものが獲得トークンです。 小数点は四捨五入されます。

{{ else }}

あなたの予想とメンバーＤの予想の差（予想誤差）を100分の1にして、二乗します。 その値に30を掛けた値を120から差し引いたものが獲得トークンです。 小数点は四捨五入されます。

{{ endif }}
